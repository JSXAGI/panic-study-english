⚡️ パニック英語学習 (Panic English Learning) 🏃‍♂️💨
「止まったら、忘れる。考えたら、パニック。直感で突き進む超実践的英語学習ツール」

文法を頭で考えるヒマを与えない！全100問のシャッフル問題に挑み、夜空に流れる星のように知識を定着させる、爆速英語学習アプリです。

🌟 特徴 (Features)
100問カオス・シャッフル: 厳選された100問を、レベルも種類も完全無視でランダム出題。脳に休む暇を与えません。

夜空の魔法演出: 正解を導き出した時、画面には美しい星空とエフェクトが。学習の疲れを視覚的に癒やします。

直感重視インターフェース: 余計な説明は省きました。見て、解いて、次へ行く。それだけのシンプル構成。

Gemini共同開発: AIの独創的なコードと、人間のパッションが融合して爆誕しました。

🚀 使い方 (Usage)
GitHub PagesのURLにアクセス。

表示される英文の空欄に入る正解を脳内で爆速で考える。

「答えを見る」を押し、自分の直感と照らし合わせる。

「次の問題（シャッフル）」で、無限の英語の海へ。

🛠️ Customize & Build Your Own (あなた独自の「パニック」を！)
このアプリのソースコードは、**「自分専用のパニック学習ツール」**を作るためのベースとして自由に使っていただけます。

問題の差し替え: script.js（またはデータファイル）内の問題リストを書き換えるだけで、単語帳、歴史の年号、あるいは難解な専門用語クイズへと瞬時に変貌します。

演出の改造: 星空のエフェクトを自分好みの演出に変更して、世界に一つだけの学習環境を構築してください。

ハック歓迎: シンプルな構造なので、プログラミング学習の練習台としても最適です。
